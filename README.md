# drewgarver.com

## How to Setup

### 1. Clone

`git clone https://github.com/ianrose/ianrose.me.git`

### 2. Install Dependencies

```
$ npm install
```

## How to use

To develop: `npm run dev`

To generate: `npm run build`

## Requirements

Node.js
